AugularJS

AngularJS简介

AngularJS 是一个 JavaScript 框架。它可通过 标签添加到 HTML 页面。

AngularJS 通过 指令，扩展了 HTML，且通过表达式绑定数据到 HTML。

eg：第一个AugularJS程序，实现计数器自增功能：

首先要在项目下新建lib文件夹，导入augular.js文件

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body ng-app>
    <input type="number" ng-model="value">
    <input type="button" ng-click="value=value+1" value="增加">
    <script src="lib/angular.js"></script>
</body>
</html>

AugularJS 指令

AngularJS 指令是以 ng 作为前缀的 HTML 属性

上面代码中用到的指令：

  • ng-app：该指令用于告诉 AngularJS ，当前这个元素是根元素。AngularJS 应用程序由ng-app定义，该元素里面是AugularJS的操作范围。原则上HTML 文档中只允许有一个 ng-app指令，如果有多个 ng-app 指令，则只有第一个会被使用。

    ng-app 和 ng-app=" “

    ng-app有一个默认的模块

    ng-app = “自定义的模块”，是一个使用了自定义的模块，该自定义的模块继承了那个初始默认的模块，可以调用初始默认的模块的方法。

  • ng-model：该指令可以将输入域的值与 AngularJS 创建的变量绑定

  • ng-click：该指令告诉了 AngularJS HTML 元素被点击后需要执行的操作

AngularJS 模块与控制器

1）模块

通过angular.module( )来创建模块

该函数有两个参数：

参数1：定义模块的名称

参数2：一个数组，存放该模块所依赖的其他模块的名称，如果确定新模块没有依赖关系，必须设置为空数组[]

注意：通过angular.module()方法定义的模块是唯一的，如果重复定义就会覆盖前面的定义

eg：定义模块，并在在html标签上启动该模块

<div ng-app="myApp"></div>
<script>
	var app = angular.module("myApp",[]);
</script>

2）控制器

控制器是JavaScript对象，由标准的JavaScript对象的构造函数创建。当控制器通过ng-controller指令被添加到页面时，AngularJS会通过控制器构造函数生成一个对象。

通过$scope对象与页面中的元素进行数据绑定，实现数据从控制器（Controller）到视图（View）的过程。可以看作是由控制器（Controller）封装后的ViewModel。

怎么用？首先得声明模块，在模块下可以在标签上挂载控制器→控制器的作用范围仅限于当前模块→所有是局部控制器

创建控制器语法：

var app = angular.module("myApp",[]);
app.controller('mainController',function ($scope) {
    //可以为$scope准备各种数据
});

.controller()函数两个参数：

参数1：控制器名称

参数2：控制器的回调函数，回调函数的参数$scope表示模块的作用域

eg：使用ng-controller指令为标签绑定控制器：

<div ng-controller="mainController"></div>

eg：AugularJS中模块和控制器用法：

定义一个模块，在模块下定义了两个控制器，使用这两个控制器

导入angular.js文件时，必须放在操作之前，否则AngularJS操作不生效

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body ng-app="app">

    <!--使用控制器1-->
    <div ng-controller="mainController1">
        <span>{{name}}</span>
    </div>

    <!--使用控制器2-->
    <div ng-controller="mainController2">
        <span>{{name}}</span>
    </div>
    <!--导入angular.js文件-->
    <script src="lib/angular.js"></script>
    <script>
        //自定义模块
        var app = angular.module("app",[]);
        
        //创建控制器1
        app.controller('mainController1',function ($scope) {
            $scope.name = "zhangSan";
        });
        //创建控制器2
        app.controller('mainController2',function ($scope) {
            $scope.name = "liSi";
        });
    </script>
</body>
</html>

3）模块间的互相依赖

模块的相互依赖可以实现功能和数据共享

eg：demo模块依赖myApp模块，demo模块可以访问myApp模块里的数据

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body ng-app="demo">

    <div ng-controller="mainController">
        <span>{{name}}</span><br>
    </div>

<script src="lib/angular.js"></script>
<script>
    var myApp = angular.module("myApp",[]);
    myApp.controller('mainController',function ($scope) {
        $scope.name = "zhangsan";
    });

    var demo = augular.model("demo",['myApp']);
</script>
</body>
</html>

eg：appA和appB模块互相依赖，数据可以共享，但是两个模块的控制器重名时，使用的模块会先从自己里找

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body ng-app="appA">

    <div ng-controller="appController">
        <span>{{name}}</span>
    </div>

<script src="lib/angular.js"></script>
<script>
    var appA = angular.module("appA",['appB']);
    appA.controller('appController',function ($scope){
        $scope.name = "zhangsan";
    })
    var appB = angular.module("appB",['appA']);
    appB.controller('appController',function ($scope){
        $scope.name = "lisi";
    })
</script>
</body>
</html>

AugularJS作用域

作用域（$scope）是控制器回调函数的参数，当一个控制器被定义时，就产生了一个作用域。在$scope作用域中定义的属性和函数，在视图（html）中可以获取使用。Scope(作用域) 是应用在 HTML (视图) 和 JavaScript (控制器)之间的纽带

根作用域$rootScope

所有的应用都有一个 $rootScope，它可以作用在 ng-app 指令包含的所有 HTML 元素中，相当于全局变量。

$rootScope 可作用于整个应用中。是各个控制器中 scope 的桥梁。用 $rootscope 定义的值，可以在各个控制器中使用

eg：$rootScope 可作用于整个应用中

<body ng-app="myApp">

    <span>{{rName}}</span>

    <div ng-controller="demo">
        <span>{{rName}}</span><br>
        <span>{{name}}</span>
    </div>

<script src="lib/angular.js"></script>
<script>
    var app = angular.module("myApp",[]);
    app.controller('demo',function ($scope,$rootScope) {
        $scope.name = "scope";
        $rootScope.rName = "rootScope";
    });
</script>
</body>
